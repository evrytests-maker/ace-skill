# Regex-шаблоны для lorebook SillyTavern

> Готовые к копированию паттерны. Настройки: Whole Words = ON, Case Sensitive = OFF.
> **Каждый ключ в lorebook ДОЛЖЕН содержать оба паттерна: EN + RU**

---

## 0. Билингвальные ключи: формат EN + RU

> Реальные примеры из загруженных пользователем лорбуков

Каждая запись lorebook должна содержать ключи на **обоих языках** в одном массиве `keys`:

### Пример из TIBIDOKHS (название школы)
```json
"keys": [
  "/\\bTibidokhs\\b/i",
  "/(?:^|[^а-яёА-ЯЁ])Тибидохс[а-яёА-ЯЁ]*(?=[^а-яёА-ЯЁ]|$)/i"
]
```
| Язык | Паттерн | Почему именно так |
|------|---------|-------------------|
| **EN** | `\bTibidokhs\b/i` | `\b` — граница слова, `i` — игнор регистра. Для английского достаточно. |
| **RU** | `(?:^|[^а-яёА-ЯЁ])Тибидохс[а-яёА-ЯЁ]*(?=[^а-яёА-ЯЁ]|$)/i` | `(?:^|[^а-яёА-ЯЁ])` — не русская буква слева, `[а-яёА-ЯЁ]*` — wildcard для всех окончаний, `(?=[^а-яёА-ЯЁ]|$)` — не русская буква справа или конец строки. |

### Пример из Gacha RPG (имя персонажа)
```json
"keys": [
  "Lucius",
  "/(?:^|[^а-яёА-ЯЁ])[Лл]юциус(?:а|у|ом|е|ы|ов|ами?|ах)?(?=[^а-яёА-ЯЁ]|$)/g"
]
```
| Язык | Паттерн | Почему именно так |
|------|---------|-------------------|
| **EN** | `Lucius` | Простое имя — SillyTavern сам обрабатывает как целое слово. |
| **RU** | `(?:^|[^а-яёА-ЯЁ])[Лл]юциус(?:а|у|ом|е|ы|ов|ами?|ах)?(?=[^а-яёА-ЯЁ]|$)/g` | `[Лл]` — оба регистра, полный набор падежных окончаний, `g` — глобальный флаг. |

### Правило: EN vs RU паттерны

| Язык | Граница слова | Склонение | Флаг регистра |
|------|--------------|-----------|---------------|
| **EN** | `\b` — стандартная | Опционально: `(?:s\|ed\|ing)?` | `/i` — case-insensitive |
| **RU** | `(?:^|[^а-яёА-ЯЁ])...(?=[^а-яёА-ЯЁ]|$)` — lookaround | Обязательно: все 6+ падежей или wildcard `[а-яёА-ЯЁ]*` | `/i` или `[Бб]` — оба регистра |

### Антипаттерн: НЕ смешивай `\b` с кириллицей
```
❌ /\bТибидохс\b/i       — \b не работает надёжно с кириллицей
❌ /\b[Лл]юциус\b/g      — \b не работает надёжно с кириллицей
✅ /(?:^|[^а-яёА-ЯЁ])Тибидохс[а-яёА-ЯЁ]*(?=[^а-яёА-ЯЁ]|$)/i
✅ /(?:^|[^а-яёА-ЯЁ])[Лл]юциус(?:а|у|ом|е|ы|ов|ами?|ах)?(?=[^а-яёА-ЯЁ]|$)/g
```

## 1. Семья (Family)

### Английские
```
/\b[Mm]other(?:s)?\b/g
/\b[Ff]ather(?:s)?\b/g
/\b[Pp]arent(?:s|al|ing|hood)?\b/g
/\b[Ff]amil(?:y|ies|ial)?\b/g
/\b[Mm]om(?:s|my|mie)?\b/g
/\b[Dd]ad(?:s|dy|dies)?\b/g
/\b[Ss]ister(?:s)?\b/g
/\b[Bb]rother(?:s)?\b/g
```

### Русские
```
/(?<![а-яёА-ЯЁ])[Мм]ам(?:(?:а|ы|е|у|ой)|у|очек|очкам|очками|очках)(?![а-яёА-ЯЁ])/g

/(?<![а-яёА-ЯЁ])[Пп]ап(?:(?:а|ы|е|у|ой)|ы|очек|очкам|очками|очках)(?![а-яёА-ЯЁ])/g
/(?<![а-яёА-ЯЁ])[Оо]тец(?:(?:а|у|ом|е)|ы|ов|ам|ами|ах)?(?![а-яёА-ЯЁ])/g
/(?<![а-яёА-ЯЁ])[Сс]емь(?:(?:я|и|е|ю|ей)|и|ям|ями|ях)(?![а-яёА-ЯЁ])/g
/(?<![а-яёА-ЯЁ])[Сс]естр(?:(?:а|е|ы|у|ой)|ы|ам|ами|ах)|сестёр(?![а-яёА-ЯЁ])/g

/(?<![а-яёА-ЯЁ])[Рр]одител(?:(?:ь|я|ю|ем|е)|и|ей|ям|ями|ях)(?![а-яёА-ЯЁ])/g
```

## 4. Шаблон: имя NPC (NPC Names)

### Английское имя
```
/\b[Nn][Aa][Mm][Ee]\b/g
```
Пример для "Alex":
```
/\b[Aa]lex(?:andra|ander|is)?\b/g
```

### Русское имя
```
/(?<![а-яёА-ЯЁ])[Ии]мя_здесь(?![а-яёА-ЯЁ])/g
```
Пример для "Анжела" (Angela):
```
/(?<![а-яёА-ЯЁ])[Аа]нжел(?:(?:а|е|ы|у|ой)|ы|ам|ами|ах)(?![а-яёА-ЯЁ])/g
```

### Имя + прозвище
```
/\b[Nn]ame\b|\b[Nn]ickname\b/g
```

## 5. Универсальный шаблон (Universal Template)

### Параметры
| Параметр | Описание | Пример |
|---|---|---|
| `[STEM]` | Основа слова без окончания | `цветочн` |
| `[ENDINGS]` | Список окончаний через `\|` | `ое\|ого\|ому\|ым\|ом` |
| `[LANG]` | Язык (en/ru) | `ru` |

### Английская версия
```
/\b[Cc][STEM](?:[ENDINGS])?\b/g
```

### Русская версия
```
/(?<![а-яёА-ЯЁ])[Пп][STEM](?:[ENDINGS])(?![а-яёА-ЯЁ])/g
```

### Примеры заполнения

**"красивый" (муж. род, полное склонение)**
```
/(?<![а-яёА-ЯЁ])[Кк]расив(?:(?:ый|ого|ому|ым|ом)|ая|ой|ую|ое|ые|ых|ым|ыми|а|ы|е|у|ой|о|ам|ами|ах)(?![а-яёА-ЯЁ])/g
```

**"happy"**
```
/\b[Hh]app(?:y|ier|iest|ily|iness)?\b/g
```

**"нежный" (мягкий знак + окончания)**
```
/(?<![а-яёА-ЯЁ])[Нн]ежн(?:(?:ый|ого|ому|ым|ом)|ая|ой|ую|ое|ые|ых|ым|ыми)(?![а-яёА-ЯЁ])/g
```

## Быстрая шпаргалка: типы склонений

| Тип | Структура | Пример |
|---|---|---|
| Мужской 1 | основа + (а\|у\|ом\|е) + мн.ч. | стол → стола, столу, столом |
| Мужской 2 | основа + (я\|ю\|ем\|е) + мн.ч. | словарь → словаря, словарю |
| Женский 1 | основа + (ы\|е\|у\|ой) + мн.ч. | рука → руки, руке, руку, рукой |
| Женский 2 | основа + (и\|е\|ю\|ей) + мн.ч. | неделя → недели, неделе |
| Средний | основа + (а\|у\|ом\|е) + мн.ч. | окно → окна, окну |
| Множественное | основа + (ам\|ами\|ах) | суток → суткам, сутками |

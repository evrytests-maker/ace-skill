# RU Lorebook Regex Checker — Справочник

Инструмент для проверки русских ключевых слов лорбука SillyTavern.

## Скрипт
`scripts/ru_regex_check.py` — CLI-утилита Python 3.

## Использование

### Линтинг regex-ключа:
```bash
python3 scripts/ru_regex_check.py '<REGEX>' --lint --kind name|word|phrase --strategy regex|full-form|stem
```

### Проверка кандидатов:
```bash
python3 scripts/ru_regex_check.py '<REGEX>' --kind name 'Кандидат1' 'Кандидат2' 'левый запрос'
```

### Автоисправления:
```bash
python3 scripts/ru_regex_check.py '<REGEX>' --fix              # JSON-экранирование
python3 scripts/ru_regex_check.py '<REGEX>' --fix-strict-space # \s* → \s+
```

## Типы ключей (--kind)

| Тип | Описание | Примеры |
|-----|----------|---------|
| `name` | Имена, фамилии, прозвища, названия | Годжо, Зенин, Токио |
| `word` | Одиночные слова, термины | проклятие, техника, барьер |
| `phrase` | Словосочетания | проклятая энергия, сборка пк |

## Стратегии (--strategy)

| Стратегия | Когда использовать |
|-----------|-------------------|
| `regex` | Точные границы с `(?:^|[^а-яА-ЯёЁ])...(?![а-яА-ЯёЁ])` |
| `full-form` | Перечисление всех форм слова без regex |
| `stem` | Обрезанная основа (только для длинных уникальных корней) |

## Шаблоны regex

### Имя персонажа (строгий регистр):
```
/(?:^|[^а-яА-ЯёЁ])Годжо(?![а-яА-ЯёЁ])/g
```

### Имя + варианты:
```
/(?:^|[^а-яА-ЯёЁ])(?:Гето|Гэто|Сугуру\s+Гето)(?![а-яА-ЯёЁ])/g
```

### Слово с падежами:
```
/(?:^|[^а-яА-ЯёЁ])[Пп]роклят(?:ие|ия|ию|ием|ии|иями)(?![а-яА-ЯёЁ])/g
```

### Слово с ё/е:
```
/(?:^|[^а-яА-ЯёЁ])[Чч][её]со(?:[а-яё]*)(?![а-яА-ЯёЁ])/g
```

### Словосочетание с пробелом:
```
/(?:^|[^а-яА-ЯёЁ])[Пп]роклят(?:ая|ой|ую)\s+[Ээ]нерги(?:я|и|ю|ей)(?![а-яА-ЯёЁ])/g
```

### Словосочетание с двумя порядками:
```
/(?:^|[^а-яА-ЯёЁ])(?:пк[-\s]+сборк(?:а|и|у|ой)|сборк(?:а|и|у|ой)\s+пк)(?![а-яА-ЯёЁ])/g
```

## Вывод скрипта

- `INFO` — детектированный тип и стратегия
- `FIX` — безопасная механическая правка
- `WARN` — риск логики (требует внимания)
- `ERROR` — ключ сломан или не RU
- `OK:` / `NO:` — результат проверки кандидата
- `SKIP non-ru:` — кандидат не русский

## Плохие ключи (запрещённые)

Слишком короткие/неоднозначные: `лес`, `пол`, `мак`, `он`, `она`, `в`, `на`

Плохие regex: `/\bпроклятие\b/i`, `/.*техника.*/i`, `/сборк\s*пк/i`
